# Solveur-de-l-quation-de-Kepler
Ce programme résout l’équation de Kepler en calculant l’anomalie excentrique E à partir de l’anomalie moyenne M et de l’excentricité e. Il utilise une méthode d’itération avec relaxation pour approcher progressivement la solution numérique. Résultat converti en radians et degrés.
